# Hospital Management System

A Java-based Hospital Management System with JavaFX GUI and MySQL database.

## Features
- Patient Management
- Doctor Management
- Room Allocation
- Database Integration (MySQL)

## Technologies Used
- Java
- JavaFX 21
- MySQL
- JDBC (MySQL Connector 9.7.0)

## Setup
1. Install Java 21+
2. Install JavaFX 21 SDK
3. Setup MySQL database
4. Run `Main.java`
